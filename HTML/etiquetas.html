¿Qué es Browser?



Un browser o navegador web, como se conoce en español, es un software, programa o aplicación que opera a través de la Web, y permite la interpretación de los contenidos varios archivos y sitios web. Esta interpretación se hace a través de la traducción del código de la página hacia un lenguaje comprensible por el usuario.

La funcionalidad básica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos multimedia incrustados. Además, permite visitar páginas web y hacer actividades en ella, es decir, enlazar un sitio con otro, imprimir, enviar y recibir correo, entre otras funcionalidades más.

Los documentos que se muestran en el navegador pueden estar almacenados localmente en el computador del usuario y también pueden estar ubicados en otro dispositivo conectado con el computador del usuario a través del internet, y que tenga todos los recursos necesarios para la transmisión de los documentos (un software servidor web).

Gracias al uso de un browser, por lo tanto, una persona puede acceder a la información de las páginas web e interactuar con ella. La navegación virtual se posibilita mediante los enlaces (también llamados links o hipervínculos) que hacen que el usuario pueda pasar de un sitio web a otro.
